33

34 
 * The check will flag the following with warnings:
35 
 * </p>
36 
 * <pre>
37 
 * return (x);          // parens around identifier
38 
 * return (x + 1);      // parens around return value
39

40

41

42

43

44

45

46 
 * </pre>
47 
 * <p>
48 
 * The check is not "type aware", that is to say, it can't tell if parentheses
49

50

51

52

53

54

63

64

66
